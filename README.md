Lab 4 & 5 - Fullstack
----

### Prerequisities

Docker

### Running

Fetch modules
`git submodule update --init --recursive`

Run the stack
`docker compose up -d`

The web app will be available on port `:3000`

API available on port `:8888`

### Live Deployment

Webapp is running on https://issi-react-lab2-web.onrender.com/

:warning: You might have to wait for API to warm up.
